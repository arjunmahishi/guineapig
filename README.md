# guineapig

A quick-generic-mock-HTTP-server to test outbound HTTP requests

## Installation

Get the executable binary from the latest release

### Compile it yourself

```bash
git clone git@github.com:arjunmahishi/guineapig.git
cd guineapig
go build -o $GOBIN/guineapig
```

## Usage

```bash
# Get help
$ guineapig -help
```
